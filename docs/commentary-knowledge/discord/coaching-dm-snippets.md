## Msven coaching Surfinite on Apollo (2024-07-03)

**Replays discussed**: `tOZQb-7kgI9`, `7nnQH-D0yTH`

### Apollo evaluation framework (Msven)
- Apollo is **bad value for money** when it only kills 3 defense or 1-damage attackers per turn
- "Option value for walls vs attackers isn't worth what's being given up"
- **12rrr for 3 Tarsiers is much cheaper** than Apollo
- Key factors for Apollo viability:
  1. Triple blue has to be useful for follow-up
  2. Opponent can't easily curb the 3 threat on defense
  3. **Any turn where Apollo only threatens 2 def is salt in the wound**

### Replay 1 (tOZQb-7kgI9) — base set + Auric/Shredder/Ossification
- "I wouldn't 3rd engi and also wouldn't buy Conduit. Just fastimus"

### Replay 2 (7nnQH-D0yTH)
- "Slightly better for Apollo because it hits Urban Sentry, but you could buy Urban every turn"
- "Apollo looks weak here because you have to buy Steelsplitter with blue"
- "You bought too many Trinities in a set that doesn't have enough breachproof supplies"
- "Probably want an Animus for Tarsiers"
- "I would buy 1-2 Trinities and save the rest of the drones for Plexo Cell"

### Key takeaway
- Supply constraints (limited breachproof supplies) can make Apollo-centric strategies fail even when Apollo looks viable
- Msven suggests investing in Plexo Cell over excess Trinities when supply is limited
