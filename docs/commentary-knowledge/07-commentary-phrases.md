# Category 7: Commentary Phrases & Terminology

## Glossary of Game Terms
> Source: Glossary (Prismata wiki) + Official Rules + PRISMATA_REFERENCE.md + Yujiri Jargon guide (archived)

### Core Mechanics Terms

| Term | Definition | Commentary Usage |
|------|-----------|-----------------|
| **Absorb** | Damage taken by a non-fragile unit that doesn't kill it — heals back next turn | "Getting great absorb on that Wall" / "Denying absorb with precise attack" |
| **Soak** | Defense that dies. A blocker sacrificed to take lethal damage | "Soaking 3 with that Forcefield" |
| **Breach** | When attack exceeds total defense — attacker targets units of choice | "That's a breach! Targeting the Drones!" |
| **Exploit** | Attacking for an amount that denies absorb due to poor granularity | "Exploiting the lack of granularity — attacking for exactly 3 against two Walls" |
| **Float** | Unspent resources at end of turn | "Floating 4 gold — that's wasteful" |
| **Overdefend** | Spending more on defense than needed | "Overdefending here — those resources could go to attackers" |
| **Overshoot** | Attack beyond absorb that damages the absorber but doesn't kill it | Used when analyzing damage efficiency |
| **Chill** | Prevents blocking. Unit with chill ≥ HP is frozen | "Freezing the Wall with Shiver Yeti — denying absorb entirely" |
| **Prompt** | Zero build time. Can block immediately | "That prompt Wall saves them from breach" |
| **Frontline** | Can be targeted by opponent even through blockers | "Frontline unit — vulnerable but threatening" |
| **Fragile** | Does NOT heal after taking damage | "Fragile means every point of damage is permanent" |
| **Lifespan** | Unit dies when counter reaches 0 | "Only 2 turns left on that Doomed Wall" |
| **Exhaust** | Turns disabled after ability use | "Exhaust 1 means it alternates — attacking every other turn" |
| **Stamina** | Number of times click ability can be used | "Two charges left on that Rhino" |
| **Buildtime** | Turns before newly purchased unit becomes active | "Under construction — invulnerable but useless for now" |

### Strategic Terms

| Term | Definition | Commentary Usage |
|------|-----------|-----------------|
| **Absorb barrier** | HP of biggest absorber in the set | "Centurion sets the absorb barrier at 5 — expect big economies" |
| **Granularity** | Ability to block any specific damage amount efficiently | "Good granularity with Wall + Engineers — opponent can't exploit" |
| **Defensive granularity** | Varied HP values across blockers to handle any attack | "Mixed blockers give excellent defensive granularity" |
| **Breach-vulnerable** | Units likely to be targeted during breach (low HP, high value) | "Tarsiers are breach-vulnerable — 1 HP each" |
| **Tech** | Conduit, Blastforge, or Animus (resource-producing buildings) | "Going for double Blastforge tech" |
| **Econ / Economy** | Drone count and gold production capacity | "Building econ to 18 Drones" |
| **Tempo play** | Aggressive strategy aiming to end quickly | "This is a tempo play — sacrificing econ for early pressure" |
| **Econ play** | Defensive strategy aiming to outscale | "Going for the econ play — big economy, big absorber, win late" |
| **Set reading** | Analyzing the random units to determine strategy | "Reading this set: big absorber available, looks like a defensive game" |
| **Breachproof** | Strategy using high-HP units to survive breach safely | "Going breachproof — all Gauss Cannons, no blockers needed" |
| **Key unit** | Win condition unit that defines the strategy | "Drake is the key unit here — massive attack potential" |
| **Support unit** | Unit that assists key units but isn't the win condition | "Engineers are support — cheap blockers for granularity" |
| **The Rhino train** | Buying 1-2 Rhinos per turn for steady defense | "Starting the Rhino train — sustainable red defense" |
| **Offensive finesse** | Using click units to force opponent guessing (click or block?) | "Steelsplitter finesse — will they click for attack or hold to block?" |

### Phases of the Game

| Phase | Description | Commentary Cues |
|-------|-------------|-----------------|
| **Opening** | Turns 1-4. Economy building, tech decisions | "Standard opening" / "Aggressive opening — Animus on turn 2!" |
| **Mid-game** | Economy established, first attackers out | "Transitioning to attackers" / "Over the absorb barrier now" |
| **Late-game** | Full army, defense critical | "It's all about defense now" / "Looking for the finishing blow" |
| **Endgame** | One side collapsing, breach imminent | "This is the endgame — defense crumbling" |

### Community Jargon (Yujiri)
> Source: Yujiri — Jargon (yujiri.xyz, archived)

| Term | Definition | Commentary Usage |
|------|-----------|-----------------|
| **BSO** | Base Set Only (no random units) | "In BSO, Wall is the only absorber" |
| **Base + X** | Base set plus listed unit(s) or number of randoms | "In base + Hellhound, Aniforge is strong" |
| **Line notation** | DD/DA = Buy 2 Drones, then Drone+Animus (slash = next turn) | "Player 2 opens DD/DDC — Natural Conduit" |
| **Autowin** | One player has an unbeatable line with optimal play | "Theoretically every set is an autowin, but execution matters" |
| **Legendary** | A unit with only 1 supply | "Odin is legendary — first to buy it has a huge advantage" |
| **Inflation** | How much more valuable things are now vs next turn | "High inflation means buy prompt units, not Drones" |
| **BV** | Barrier Value — gold cost of 1 point of defense | "Low BV means defense is cheap, so econ is favored" |

## Color Commentary Vocabulary
> Source: Synthesized for natural-sounding casts

### Positive / Exciting
- "Crisp play" — optimal move with minimal waste
- "Clean defense" — exactly enough blocking, maximum absorb
- "Beautiful read" — correctly anticipated opponent's strategy from the set
- "Explosive turn" — high burst damage or many purchases in one turn
- "Efficient tech path" — reaching key units without wasting resources

### Negative / Concerning
- "Wasteful float" — too many resources unspent
- "Overcommitted" — too much invested in one direction
- "Breach-vulnerable" — key units exposed to breach targeting
- "Overdefending" — spending defense resources that could be attackers
- "Misread the set" — built wrong strategy for available units

### Neutral / Analytical
- "Standard development" — following Standard Style principles
- "Tech-heavy" — many tech buildings, expensive to maintain
- "Resource-efficient" — minimal waste per turn
- "Position is roughly even" — neither side has clear advantage
- "The race is on" — both sides attacking, who dies first?
