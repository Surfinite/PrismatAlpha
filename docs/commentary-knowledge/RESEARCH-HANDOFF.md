## Important Notes About the Game

- **Prismata is NOT**: a card game with randomized draws, a mobile game, anything involving "prisms" or "prisma"
- **Prismata IS**: a deterministic turn-based strategy game with perfect information. No RNG during gameplay (only the 8 random units selected before the game starts). Two players, asymmetric starts (P1 goes first, P2 gets an extra Drone).
- **The game launched** on Kickstarter Oct 2014, Steam Early Access Nov 2017. Servers are still running but the player base is very small.
- **Key community figures**: Elyot Grant and Will Ma (developers/Lunarch Studios), 307th/Arkanishu (#1 player, strategy guide author), Yujiri (former #2, math-focused guides), Msven (top player, creative playstyle), amalloy (community contributor, video series), RuinedShadows (Master in a Month guide), swol/Silverbean (top ranked players)
- **Internal codenames**: The engine uses codenames (Tesla Tower = Tarsier, Brooder = Blastforge, etc.). If you find content using these names, it's real Prismata content.
- **Common false positives in search**: Cosmoteer ("Guide to Prisms"), Banner Saga ("Combat Strategy Primer"), Slime Rancher 2 ("Prisma Plorts"), military strategy articles, business strategy posts

## How to Verify a Source is About Prismata

A real Prismata strategy source will mention at least several of these terms: Drone, Wall, Tarsier, Blastforge, Conduit, Animus, Engineer, Forcefield, Steelsplitter, Rhino, Gauss Cannon, absorb, breach, chill, breachproof, soak, granularity, exploit.

If a source mentions "Prismata" but none of these unit/mechanic names appear, it's probably about a different game or a general article.
