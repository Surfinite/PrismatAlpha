= Prismata =

{{stub}}
'''''Prismata''''' is a strategy game by [[Lunarch Studios]] borrowing elements from various genres, such as real-time strategy games to create a hybrid experience. Like a collectible card game, it uses turn-based gameplay, but it doesn't involve creating a deck. Like a real-time strategy games, it relies heavily on build orders and tactics, but there is no map. It is fast-paced with games lasting only minutes.
