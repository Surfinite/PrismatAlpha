= Random set =

{{stub}}
The '''Random set''' is composed of Random Units which includes all [[Units]] that are not part of the [[Basic set]]. A Random set is made of five or eight of the Random Units (base+5 or base+8).
The ability to read the Random set of a game is a must for a player for it determines the best strategy to be be set in motion.
For instance, if the Random set contains a lots of defenders there is no need to value a rushy strategy over a blue high [[Economy|economical]] one.
